Web Application - Challenge 1: Plume display

Designed by Romain Boureau

This web application was designed for the technical challenge : plume display.
There are two parts to this challenge : a Restful API and a web application.
Please find below these two parts.

This part can be used to test the Restful API and understand how to use it.
The URI is http://127.0.0.1:5000/api/image and it uses the POST method.
As you can see in this example, three arguments are needed : latitudeH and longitudeH are obviously the latitude and longitude of the image from the Google maps API, fileupload is the png image which will overlay the other.


This part is the web application displaying the overlaid image according to the browser width.
Three arguments are needed : latitude and longitude are coordinates for the Google Maps API, fileupload is the png image which will overlay the other.
